    Velin

    Velin is a lightweight, runtime-only frontend reactivity engine designed for maximum flexibility and zero compilation. It enables expressive, declarative UIs directly in HTML using custom vln-* directives, and integrates seamlessly with vanilla JavaScript.

    Example

    Here's a minimal interactive playground using Velin + Tailwind (no build tools):

    <script src="velin.js"></script>
    <input vln-input="vln.name" />
    <h1 vln-text="'Hello ' + vln.name"></h1>
    <script>
      const vln = Velin.bind(document.body, {
        name: "Andrew",
      });
    </script>

    Velin binds DOM elements to live state objects with minimal overhead and zero boilerplate.

    Usage

    Just include velin.js in your HTML. No npm, no bundler, no fuss.

    <script src="velin.js"></script>

    Initialize with:

    const vln = Velin.bind(document.body, {
      ... your_state_properties
    });

    Benchmarks

    The benchmarks/ folder contains real-world implementations using:

    • React
    • Angular
    • Alpine.js
    • Velin

    To build and serve all benchmark apps:

    npm run prepare:benchmarks
    npm run serve:benchmarks

    Use these to compare bundle size, rendering speed, and developer effort.

    Testing

    Velin uses Vitest for lightweight test execution and JSDOM for DOM simulation. Visual snapshot testing is handled manually through Playwright or similar tools (planned).

    To run tests:

    npm test
