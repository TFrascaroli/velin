Learn Velin

You'll be reactive in about fifteen minutes. Velin has no compiler, no special file format, and nothing to get past.

The 60-second install

Drop a script tag, write a plain object, call bind:

<script src="https://unpkg.com/@velinjs/all/velin-common.min.js"></script>

<div id="app">
  <h1 vln-text="'Hello, ' + name + '!'"></h1>
  <input vln-input="name" />
</div>

<script>
  Velin.bind(document.getElementById('app'), { name: 'World' });
</script>

The five things to learn

  1. Bind state. Velin.bind(root, state) attaches a plain JavaScript object to a DOM subtree. Assigning to a property re-renders the parts of the DOM that read it. Getting Started →
  2. Use directives. Nine everyday ones cover 95% of UIs: vln-text, vln-if, vln-attr, vln-class, vln-on, vln-input, vln-watch, vln-loop, vln-use. Directives reference →
  3. Loop over lists. vln-loop:item="items" repeats an element per array entry. Nested loops compose naturally. CRUD example →
  4. Compose with templates. Repeated markup lives in a <template>; mount it with vln-fragment and pass variables via vln-var:name. Templates guide →
  5. Extend with plugins. Every directive is a plugin. Write your own with the same API the built-ins use. Plugin authoring →
