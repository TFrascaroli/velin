VS Code Extension

LSP-backed IntelliSense for vln-* directives — hover docs, completion inside directive expressions, diagnostics for unknown directives and malformed expressions.

What it gives you

  • Autocomplete for every built-in directive name and subkey.
  • Expression completion inside directive attribute values, driven by the LSP.
  • Hover documentation for each directive.
  • Diagnostics for unknown directives, malformed expressions, and missing state.

Installing (local build for now)

The extension isn't on the Marketplace yet. To try it, build and install from source: 

cd tooling/velin-language-support
npm install
npm run build
npm run package:vscode
# then in VS Code: Extensions → ... menu → Install from VSIX
